Absolute/Sign
#################

abs(*x)
*****************

Absolute value of `x` ::

    abs(-12.34) //--> 12.34
    abs(0)      //--> 0
    abs(56.78)  //--> 56.78

mag(x[]...)
*****************

Magnitude of vector `x` ::

    mag(3,4)   //--> 5
    mag(3,4,5) //--> 7.071067812

sign(*x)
*****************

Returns 1 for positives, -1 for negatives, 0 otherwise. ::

    sign(-12.34) //--> -1
    sign(0)      //--> 0
    sign(56.78)  //--> 1
