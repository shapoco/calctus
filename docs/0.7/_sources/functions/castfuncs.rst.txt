Cast
#################

array(x)
*****************

Converts value `x` to an list. ::

    array([1,2,3]) //--> [1, 2, 3]
    array("abc")   //--> ['a', 'b', 'c']

rat(*x)
*****************

Rational fraction approximation of `x`.

rat(*x, max)
*****************

Rational fraction approximation of `x`.

real(*x)
*****************

Converts the `x` to a real number.

str(x)
*****************

Converts `x` to a string.
